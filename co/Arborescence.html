Arborescence classique de Windows

Fondamental :

Un dossier est un élément qui peut contenir des fichiers (documents) ou d'autre dossiers. Une représentation hiérarchique peut être utilisée pour expliquer le mode de rangement des documents dans Windows.

Une arborescence classique de Windows est décrite ici. Dans cette arborescence, certains éléments sont virtuels et d'autres des périphériques physiques.

Un élément est virtuel s'il se réfère à un emplacement physique.

Normalement votre poste se présente comme suit. C'est la configuration de l'ensemble des postes que vous retrouverez quelque soit le poste que vous utiliserez.

Si vous sélectionnez "ordinateur" dans le volet de navigation vous obtiendrez la liste des disques disponibles à partir de l'ordinateur.

Cette liste est divisée en trois :

  • disques durs

  • périphériques utilisant des supports de stockage amovibles

  • emplacement réseau

Disques durs

Les disques durs sont les périphériques intégrés au boîtier de l'ordinateur utilisés pour stocker des données.

A l'IUT vous ne devez rien stocker sur ces disques durs.

Supports amovibles

Les supports amovibles sont les clefs USB ou les lecteurs DVD. L'inconvénient des clefs USB est que l'on a tendance à les oublier ou à les perdre.

Emplacement réseau

Un emplacement réseau permet d'accéder à des données ou des programmes d'un autre ordinateur. Ce n'est pas Internet!.

Pour les étudiants de l'I.U.T. G.E.A. trois emplacements réseaux nommés H, T et L sont disponibles. Physiquement ils sont situés sur un ordinateur dans un local du bâtiment administratif.

L'emplacement H ou lecteur H contient vos travaux personnels.

Le lecteur T ( comme Tous)est un espace où vous pouvez échanger des données (accès en lecture et écriture).

Le lecteur L (comme Lecture) un espace où vous pouvez trouver des documents fournis par les enseignants.

Ces données ne sont pas accessibles par le WEB.

Attention :

Vous ne devez pas sauvegarder de document sur le bureau.

Fondamental : Représentation arborescence des dossiers

L'organisation de l'accès aux données peut être représentée par une arborescence.

Par exemple cette arborescence signifie que pour accéder au dossier programme il faudra cliquer sur "ordinateur" puis sur "c :" et enfin programmes.
